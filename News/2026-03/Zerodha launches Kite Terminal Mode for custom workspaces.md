Introducing Kite Terminal Mode 
Most trading platforms look the same for everyone. The same layout, the same widgets, and the same workflow, whether you place one trade a month or fifty trades a day.
But each trader works differently. Some prefer having charts and the order window side by side; others track multiple watchlists and the option chain simultaneously, and many want their positions, P&L, and orders visible without constantly switching tabs.
Kite Terminal Mode is designed to solve this by letting you build a workspace around how you trade. Instead of adapting to a fixed layout, you can arrange charts, watchlists, option chains, orders, and other tools to fit your workflow.
If you’d like to see how it works, watch the walkthrough below.
 
 Try Terminal Mode 
Start with a layout, or build your own
Terminal Mode allows you to create multiple workspaces, each designed for a different workflow.
You can start from a blank canvas or choose a preset layout:
Trade: A balanced layout with charts and order tools
Scalper: Built for speed and quick execution
Invest: Designed for long-term tracking
Simple: A minimal layout
General: A workspace with informational widgets
 
These layouts are only starting points. You can rearrange panels, resize widgets, or remove anything you don’t need.
Keep everything you need in one view
One of the biggest advantages of Terminal Mode is that it allows you to keep multiple parts of your trading workflow visible at the same time.
For example, you can place charts in the centre for analysis, the option chain beside it for strategy building, market watchlists on the left, and orders or positions at the bottom. This makes it easier to monitor markets and manage trades without switching between tabs or opening multiple windows.
 
Built-in tools for trading and analysis
Terminal Mode includes a wide range of widgets that help with market tracking, analysis, and execution.
 
Core trading tools include market watch, charts, the option chain, orders, trades, and positions. You can also create baskets for multi-leg strategies, place long-validity GTT orders, and set alerts based on price or volume movements.
All these widgets can be placed anywhere in your workspace, depending on how you prefer to track the markets.
In addition to trading tools, Terminal Mode also includes several utility widgets that traders often need during market hours. These include the margin calculator, brokerage calculator, economic calendar, corporate actions tracker, and notes for maintaining trading journals or recording trade ideas.
There are also widgets like the world clock for tracking global market sessions and timers that help structure trading sessions and breaks.
 
Keeping these tools inside the workspace reduces the need to switch between apps or browser tabs.
Create different workspaces for different strategies
Terminal Mode lets you maintain multiple workspaces, allowing you to organise your setup around different strategies.
For example, you might create one workspace focused on intraday trading, with charts taking up most of the screen, another for options trading, with the option chain at the centre, and a separate workspace for tracking long-term investments and alerts.
You can also generate a shareable link to your customised workspace so other traders can access it. The link can be disabled at any time if you no longer want it to be accessible.
Terminal Mode is designed to make Kite more flexible. Instead of adapting to a fixed interface, you can build a workspace that reflects how you analyse markets, track positions, and execute trades.
