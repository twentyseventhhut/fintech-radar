[Connecting the Dots in Fintech] 🇵🇱 Brighty launches an API for agentic banking. The API enables AI agents to check balances across all accounts, send payments, convert currencies in real time, manage payroll, and reconcile transactions. In addition, Brighty’s AI agent can read incoming invoices and initiate international transfers with full audit trails and compliance.
