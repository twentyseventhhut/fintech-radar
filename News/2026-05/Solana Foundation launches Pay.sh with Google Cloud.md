Access and pay-as-you-go for any API with stablecoins on Solana
AI is now core to how we build and consume software. Agents search, synthesize, decide, and act often across dozens of services in a single workflow. But there's a clear gap as the most powerful APIs still require a human to create accounts, manage credentials, KYC, and maintain billing relationships and subscriptions.
Today that changes.
In collaboration with Google Cloud, we’re introducing Pay.sh , a gateway service designed to bridge the gap between autonomous agents and enterprise infrastructure.
 Introducing Pay.sh 
Agents can now instantly discover, access, and pay-per-request for any API autonomously, for the first time supporting APIs from Google Cloud including Gemini, BigQuery, Gemini Enterprise Agent Platform, and more using stablecoins on Solana. No accounts. No API keys. No subscriptions needed. 
 “Agentic payments are one of the most important frontiers in the agentic stack. Agents need to transact autonomously, without setup or credentials getting in the way. x402 gives us the standard to do that, and Pay.sh is the first to make it real at scale. Solana was the right choice for settlement, and this is exactly the kind of infrastructure Google Cloud wants to be part of." — Rich Widmann, Head of Strategy, Web3, Google Cloud
 How does it work? 
 
Pay.sh starts with a Solana wallet linked to your favorite AI interface including Gemini, Claude Code, Codex, Openclaw, Hermes, and more. Onramp funds with a credit card or stablecoin in 60 seconds, and your agent is ready to access paid services instantly.
Agents can browse a unified marketplace of API endpoints, receive a live rate, and pay directly from its balance. No billing account to create and no subscriptions required. The payment is the credential - it’s that simple.
Under the hood, Pay.sh operates as an API proxy built on Google Cloud, sitting in front of Google Cloud services including BigQuery, Gemini, Cloud Run, and more, handling account-less payment settlement on your behalf. Your agent's Solana wallet acts as its identity with no need to create a Google account or manage rotating credentials. When an agent pays via x402, the gateway service authorizes the request through a verified endpoint, applying appropriate rate limits, quotas, and access controls so enterprise security and compliance are never compromised.
Settlement happens in seconds. Payments are processed in stablecoins over Solana's high-throughput network, then reconciled with the provider. Developers pay only for what they use, and providers receive funds reliably, with no manual billing overhead on either side.
 What can you access? 
For the first time, agents have a single place to discover, access, and pay for the APIs they need including official Google Cloud APIs plus 50+ Community API facilitators with access to services across ecommerce, market data & intelligence, communications, and onchain infrastructure - all payable per call, with no account, no subscriptions.

 Key Benefits 
 Discovery : A unified, searchable catalog that indexes every approved API endpoint allowing agents to find and call what they need in a single workflow without switching tools or hunting across providers
 Access: Tier-1 APIs have traditionally required manual credentialing. Pay.sh acts as a Gateway as a Service , providing a compliant abstraction layer that handles high-velocity micro-payments, so agents can interact with enterprise services programmatically and simply per request with no setup or credentials required.
 Pay per request: No subscriptions. No minimum spend. Your agent consumes exactly what it needs, when it needs it, and nothing more.
 Built on open standards 
 Pay.sh is built on x402 and MPP, machine-native payment protocols designed for agent-to-API commerce. We believe in supporting open rails that any developer, protocol, or facilitator can extend and build on. Payment is handled automatically under the hood, settled in stablecoins on Solana, and paid to providers in fiat. Pay.sh is also fully open source. Explore the code, contribute, or build your own integration on GitHub .
 Supercharged by the Solana ecosystem 
 Pay.sh is a coordination layer, and works together with existing infrastructure providers across the Solana ecosystem. The discovery catalog indexes services already live across Solana's agent facilitators giving the services broader reach and offering developers one place to start.
